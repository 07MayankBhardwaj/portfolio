Load-Js : Multi thread processing

ClusterLoad is a powerful and efficient load balancing tool for Node.js applications. It uses Node.js cluster module to distribute incoming traffic across multiple worker processes, providing improved performance, reliability and scalability for your applications.

The Node.js cluster module is a built-in module that allows you to create child processes (workers) to handle incoming requests in a multi-core system. It helps distribute the workload across multiple CPU cores, improving performance and scalability of your Node.js applications. Load balancing in Node.js typically involves creating a cluster of worker processes and routing incoming requests to these workers.

* Automatic distribution of incoming requests across multiple worker processes.

* Support for multiple CPU cores for efficient multi-threading.
